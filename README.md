# WorldWise App 

## Run app
> npm run dev

## Lunch Fake api 
> npm run server

## Param fake api

Modify port and delay if u need in *package.json* \
Modify data if u need in *data/cities.json* \
Consult data on *http://localhost:9000/cities*

```
    "server": "json-server --watch data/cities.json --port 9000 --delay 500"
```
